# 기온, 날씨에 따른 옷 스타일을 알려주는 패션 웹사이트

## 📀 프로젝트 내용

- 기온, 날씨에 따른 스타일을 알려주는 페이지
    - GPS를 활용하여 현재 위치를 받아온다
    - Weather API를 활용하여 3일치 날씨 예보를 받아온다
    - 해당 날씨에 따른 옷차림을 추천한다
- 자신의 데일리룩을 업로드 하는 이미지 게시판
    - 이미지 게시판 조회
    - 자신의 스타일 업로드
